# ML_Regression_Stock_Price_Prediction
Ce projet implémente un modèle de prédiction du prix des actions basé sur l'architecture Long Short-Term Memory (LSTM), un type de réseau de neurones récurrents (RNN) particulièrement efficace pour les données séquentielles comme les séries temporelles financières. Le modèle est développé en utilisant la bibliothèque PyTorch.
